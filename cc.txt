add uv and uv_installer modules; rewrite bes_project to use uv

lib/bes/uv/ (new module — programmatic uv venv management):
  uv_error.py: exception class with message and status_code
  uv_exe_info.py: namedtuple(exe_path, version) returned by uv_exe.info()
  uv_exe.py: locate the uv binary via explicit path, UV env var, ~/.local/bin/uv,
    or shutil.which; find() raises uv_error if not found; version() parses uv --version
  uv_venv.py: create and introspect a uv virtualenv; create() is idempotent; exposes
    python_exe, bin_dir, site_packages_dir as cached_property; activate_script(variant)
    returns the path to the venv activate script (supports fish/csh/bat variants)
  uv_project_options.py: bcli_options with uv_exe, python, root_dir, uv_cache_dir,
    verbose, debug; resolve_root_dir() defaults to cwd/UV_PROJECT_ROOT;
    resolve_uv_exe() delegates to uv_exe; uv_cache_dir sets UV_CACHE_DIR env var
  uv_project.py: high-level API analogous to pip_project; constructor has no side effects;
    ensure_ready() creates the venv; install(), install_requirements() (with sha256
    checksum cache to skip redundant reinstalls), installed(), outdated(), upgrade(),
    call_uv(), call_program(), program_path(), env property (sets UV_CACHE_DIR when
    uv_cache_dir is configured)
  uv_project_command_options.py, uv_project_command_handler.py,
  uv_project_command_factory.py: bcli CLI plumbing for all project operations

lib/bes/uv_installer/ (new module — bootstrap the uv binary itself):
  uv_installer_error.py: exception class
  uv_installer_options.py: bcli_options with install_dir, install_script, version,
    verbose, dry_run; resolve_install_dir() defaults to ~/.local/bin (Unix) or
    %USERPROFILE%\.local\bin (Windows)
  uv_installer_base.py: ABC with abstract install(), uninstall(), is_installed(),
    installed_version(), exe_path()
  uv_installer_macos.py, uv_installer_linux.py: download install.sh via curl then run
    with sh; support offline mode via options.install_script; set UV_VERSION and
    UV_INSTALL_DIR env vars; warn if install_dir not on PATH after install
  uv_installer_windows.py: run install script via powershell -ExecutionPolicy ByPass;
    support local script via -File; exe candidates are uv.exe and uvx.exe
  uv_installer.py: dispatcher keyed on host platform; available_installers() classmethod
  uv_installer_command_options.py, uv_installer_command_handler.py,
  uv_installer_command_factory.py: bcli CLI plumbing for install/uninstall/is_installed/
    installed_version/exe_path commands

    root_dir/projects/<name>/<version>/  uv_project root_dir for each Python version

  falls back to uv_installer into .uv_install/); _make_uv_project() replaces
  _make_pip_project(); version resolution logic preserved with a TODO to replace
